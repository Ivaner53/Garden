## Описание
Мобильное приложение для учета растений в саду и огороде. Позволяет вести каталог растений, получать напоминания об уходе и отслеживать прогресс роста.

## Установка
Для установки скачайте файл .apk из раздела Releases. Найдите файл в файловом менеджере устройства и нажмите на него. Подтвердите установку в открывшемся окне.
Системные требования: Android 7.0 и выше.

## Использование

Приложение "Сад и огород" предоставляет следующие возможности:

## Основные функции:
· Учет растений - создание базы всех растений
· Напоминания - уведомления о поливе и уходе
· Отслеживание прогресса - фотоотчеты и статистика
· Отчеты - формирование статистики по растениям
## Работа с приложением:
· При запуске открывается главный экран с растениями
· Через боковое меню осуществляется навигация
· Для каждого растения можно настроить график ухода
· Данные хранятся локально на устройстве
## Разработка:
Данный проект разрабатывается в рамках производственной практики и предназначен для образовательных целей. Проект создаётся одним разработчиком и не предусматривает привлечение сторонних участников. 
## Технические детали:
· Язык программирования: Kotlin
· Среда разработки: Android Studio
· Минимальная версия Android: 7.0 (API 24)
